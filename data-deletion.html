What Data Will Be Deleted?

When you request data deletion, we will permanently remove:

  • Profile Data: Name, profile picture, bio, location, skills

  • Communications: Messages and correspondence
  • App Usage Data: Analytics and usage patterns

How to Request Data Deletion

  4. Confirmation: We'll send you a confirmation email before processing
  5. Processing: Data deletion will be completed within 30 days
