The favicon is already set up in your `index.html`:

## Converting to ICO

To create a proper ICO file:
1. Visit https://convertio.co/svg-ico/
2. Upload `favicon-hourbank.svg`
3. Download as `favicon.ico`
4. Replace the existing favicon.ico
